# C# Basic Algorithms

Bu repo, C# kullanarak geliştirdiğim temel algoritma ve programlama örneklerini içermektedir. Amaç, programlama mantığını geliştirmek ve farklı problem türlerine çözüm üretmektir.

## İçerik

Bu repo içerisinde aşağıdaki örnekler bulunmaktadır:

* Taksimetre ücret hesaplama
* Uçak bileti fiyat hesaplama (mesafe, yaş ve yolculuk tipine göre)
* 3'ün kuvvetlerini bulma
* Faktöriyel hesaplama
* Asal sayıları bulma
* EBOB ve EKOK hesaplama

## Kullanılan Teknolojiler

* C#
* .NET Console

## Amaç

Bu çalışmalar, algoritma kurma becerisini geliştirmek, döngüler ve koşul yapıları üzerinde pratik yapmak amacıyla hazırlanmıştır.

## Not

Kodlar eğitim ve pratik amaçlı yazılmıştır. Daha karmaşık projeler için temel oluşturmaktadır.
